## 背景

需要在 CLI 本地存储加密有用户数据，因此需要一个数据库用来存储数据。

可选的方案有

 - SQL 型数据库。如 SQLite 3，优点是可以用于其它应用。
 - NoSQL 型数据库。轻量级使用，存储一定的泄露的可能性。

在 Node.js 里，NoSQL 数据库有：

 - https://github.com/typicode/lowdb
 - http://lokijs.org/#/
 - https://pouchdb.com/
